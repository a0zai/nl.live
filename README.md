## 🚀 Key Features

* **🛰️ GPS Radar System:** Geautomatiseerde straat- en zone-detectie. Alleen toegang als je fysiek aanwezig bent in een actieve zone.
* **📹 Live Broadcasting:** Deel je omgeving in realtime met de "GA LIVE" functie. High-speed frame streaming via Firebase.
* **💬 Real-time Chat:** Ultrasnelle communicatie met een ingebouwd "Typing..." systeem.
* **🆘 SOS Noodsysteem:** Verstuur direct een noodsignaal naar alle gebruikers in jouw zone met één klik.
* **👥 Live Viewer Counter:** Zie precies wie er online is in jouw specifieke straat of wijk.

## 📖 Hoe het werkt

1.  **Authenticatie:** Voer je unieke NL-ID in (bijv. NL-772).
2.  **Radar Scan:** De app scant je GPS-coördinaten en berekent de afstand tot de dichtstbijzijnde actieve zone.
3.  **Toegang:** Zodra je binnen de straal van **309 meter** bent, wordt de zone ontgrendeld.
4.  **Interactie:** Chat, bekijk live streams van buren, of start je eigen broadcast.

## 🔒 Beveiliging & Privacy
- **Geen Data Opslag:** Berichten en locaties zijn realtime en worden niet permanent bewaard voor commerciële doeleinden.
- **GPS Verified:** Voorkomt "spookgebruikers" van buiten de regio.
- **Anoniem:** Geen account nodig, volledige privacy via Firebase Anonymous Login.
